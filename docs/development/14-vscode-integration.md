## 開発効率化設定

```json
// settings.json内
"tailwindCSS.includeLanguages": {
  "typescript": "typescript",
  "typescriptreact": "typescriptreact"
},
"tailwindCSS.experimental.classRegex": [
  "tw`([^`]*)`",
  ["clsx\\(([^)]*)\\)", "(?:'|\"|`)([^']*)(?:'|\"|`)"]
]
```

```json

"typescript.preferences.includePackageJsonAutoImports": "on"
```
